# UX Research Portfolio

A static portfolio website showcasing UX research case studies, methods, research operations, and contact information.

## Overview

This portfolio presents selected UX research work by Jannis Düster, with a focus on:

- usability testing
- workflow analysis
- complex and data-heavy systems
- enterprise and B2B software
- research synthesis and stakeholder communication

The site currently includes:

- a homepage with featured case studies
- an about page with methods, tools, and working approach
- a detailed case study on NetLogCom, a usability improvement project in environmental monitoring software
